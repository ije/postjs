
# AlephJS
A radical new Front-End Framework in [deno](https://deno.land), inspired by [next.js](https://nextjs.org) and [snowpack](https://www.snowpack.dev).

- Zero Config
- Typescript with Deno
- ES Module Ready
- HMR with Fast Refresh
- SSR and SSG
- File-system Routing
- API Routes
- Built-in CSS Support
